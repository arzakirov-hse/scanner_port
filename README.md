# Система обнаружения новых открытых портов на базе Masscan

## Описание проекта

Программное решение предназначено для автоматизированного мониторинга открытых сетевых портов в заданном диапазоне IP-адресов. Система использует утилиту Masscan для высокоскоростного сканирования сети, определяет обнаруженные сервисы по номеру порта и баннеру приложения, сохраняет результаты предыдущих сканирований и уведомляет администратора о появлении новых открытых портов.

Проект разработан на языке Python и соответствует требованиям задания по разработке программного средства обнаружения новых сетевых сервисов.

---

## Основные возможности

* высокоскоростное сканирование сети с использованием Masscan;
* обнаружение открытых TCP-портов;
* асинхронный сбор баннеров обнаруженных сервисов;
* определение типа сервиса по номеру порта и содержимому баннера;
* сохранение результатов сканирования в файл;
* сравнение текущих результатов с предыдущими;
* обнаружение новых открытых портов;
* уведомление администратора через Telegram;
* уведомление администратора по электронной почте;
* автоматическое создание конфигурационного файла при первом запуске.

## Принцип работы программы

Работа программы состоит из следующих этапов:

1. Загрузка конфигурации.
2. Проверка наличия утилиты Masscan.
3. Выполнение сетевого сканирования.
4. Получение списка открытых портов.
5. Асинхронное получение баннеров сервисов.
6. Определение типа обнаруженного сервиса.
7. Сравнение результатов с предыдущим запуском.
8. Выявление новых открытых портов.
9. Отправка уведомлений администратору.
10. Сохранение текущих результатов.

## Формат конфигурационного файла

Пример файла config.json:

```json
{
    "network": "192.168.1.0/24",
    "ports": "21,22,25,80,110,143,443,445,3306,5432,6379,8080",
    "rate": 1000,
    "masscan_path": "masscan",
    "banner_timeout": 2.0,
    "concurrency": 200,
    "results_file": "results.json",
    "notifiers": {
        "telegram": {
            "enabled": false,
            "token": "",
            "chat_id": ""
        },
        "email": {
            "enabled": false,
            "smtp_host": "",
            "smtp_port": 465,
            "username": "",
            "password": "",
            "from_addr": "",
            "to_addrs": [],
            "use_tls": true
        }
    }
}
```

## Установка

### Установка Python

Установить Python версии 3.10 или выше.

Проверить установку:

```bash
python --version
```

---

### Установка Masscan

Linux:

```bash
sudo apt install masscan
```

Windows:

Скачать Masscan с официального репозитория проекта и указать путь к исполняемому файлу в параметре masscan_path.

## Запуск программы

Запуск осуществляется командой:

```bash
python scanner.py
```

После запуска программа автоматически:

* загрузит настройки;
* выполнит сканирование сети;
* определит сервисы;
* сохранит результаты;
* отправит уведомления при обнаружении новых портов.

---

## Обнаруживаемые сервисы

Программа поддерживает определение следующих сервисов:

| Порт | Сервис     |
| ---- | ---------- |
| 21   | FTP        |
| 22   | SSH        |
| 25   | SMTP       |
| 53   | DNS        |
| 80   | HTTP       |
| 110  | POP3       |
| 143  | IMAP       |
| 443  | HTTPS      |
| 445  | SMB        |
| 3306 | MySQL      |
| 3389 | RDP        |
| 5432 | PostgreSQL |
| 6379 | Redis      |
| 8080 | HTTP       |

Также используется анализ баннеров для более точного определения приложения.
